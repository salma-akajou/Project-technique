# Structure du Projet

La structure des branches et dossiers du dépôt :

- **main** : Utilisé pour l'étape finale du déploiement.
- **develop** : Pour la réalisation et la version finale du projet technique.
- **prototype** : Le code de l'exercice pour le live coding.
- **liveCoding-start** : Contient les données et les services du projet, utilisé comme point de départ pour le live coding.
- **liveCoding** : Pour l'exercice utilisé pour le live coding.
